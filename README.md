# Backend Service of Royal Threading and Beauty

Hey, there! Welcome to the backend service of Royal Threading and Beauty.

## Installation

1. First clone the repository.
   ```bash
   git@github.com:trizen-solutions/royal-shape-backend.git
   ```
2. Get the environment variables from any of collaborators
3. Run the following command to install packages
   ```bash
   yarn install
   ```
4. Now run the server in development mode
   ```bash
   yarn dev
   ```
